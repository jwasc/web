Where to Save Files

SimpleNote and DropBox
SimpleNote is best for Markdown input. It will be available everywhere, you can use their markdown viewer and it does not alter your text. The markdown support is not robust, but it is good enough. Open source. Advise always using the web version--the rendering is better.
DropBox is where I am saving all Markdown files (so, two places, SimpleNote and DropBox). The DropBox files can be used anywhere.
